My Projects

Candy Crash Game

A ReactJS game where players match and crush candies that are next to each other. When a candy is crushed, the candies above it takes their place, and new ones appear at the top. The score increases by one for each crushed candy, and the final score is shown to the user. The project was built using React Js.

Plant Disease Detection using Python

A web app that allows users to upload plant images and check if the plant is healthy or has early or late blight. The front end was built with React JS, and the back end was developed using FastAPI. A convolutional neural network model was used, achieving 95% accuracy. The project was built using Python, JavaScript, HTML, and CSS.

My Projects

ChatAway

ChatAway is an web application where users can send text messages and photos to our friends. Users can change the theme of the application and upload their own profile picture. Users can login with their Google account plus create an account. The front end was built with React.js and the back end was developed using Node.js and Express.js. The application uses Socket.io for real-time communication and MongoDB for data storage. DaisyUi and Tailwind CSS was used to style the application. The application is still in progress.

Carnival Game in VR

VR Carnival Experience is a fun virtual reality game that brings the excitement of a real carnival to life. Using a VR headset and motion controllers, players can walk around a colorful carnival, play classic games like Can Knock Down and Balloon Pop, and earn points. The carnival is full of bright colors, lively sounds, and a happy atmosphere to make players feel like they are really there. Motion controls let players throw balls and interact with the games just like in real life. Built with Unreal Engine 5.5, the game is easy to play and enjoy, even for people new to VR. The focus is on having fun, feeling part of the carnival, and enjoying a lively digital world.

Plant Disease Detection using Python

A web app that allows users to upload plant images and check if the plant is healthy or has early or late blight. The front end was built with React JS, and the back end was developed using FastAPI. A convolutional neural network model was used, achieving 95% accuracy. The project was built using Python, JavaScript, HTML, and CSS.

Candy Crash Game

A ReactJS game where players match and crush candies that are next to each other. When a candy is crushed, the candies above it takes their place, and new ones appear at the top. The score increases by one for each crushed candy, and the final score is shown to the user. The project was built using React Js.
